目标领域（与任一领域相关即保留）：
- **gorilla file system**: 类 Unix 文件系统：目录导航、文件/目录的创建读写删除、cat/cd/cp/diff/du/echo/find/grep/ls/mkdir/mv/pwd/rm/rmdir/sort/tail/touch/wc 等基础操作。
- **math api**: 数学运算与统计：加减乘除、绝对值、对数、幂、开方、取整、最值、均值、标准差、求和、百分比、单位换算（英制/SI）。
- **memory kv**: 键值记忆：长期/短期记忆的增删改查、按 key 检索、BM25+ 相似 key 搜索、清空。
- **memory rec sum**: 递归摘要记忆：对记忆内容追加、替换、检索、整体更新或清空。
- **memory vector**: 向量记忆：条目增删改查、按相似度检索（top_k）、长期与短期记忆。
- **message api**: 工作区消息与联系人：添加联系人、收发/删除/搜索消息、用户列表、登录、查看发送记录与统计。
- **posting api**: Twitter 发帖与社交：认证、发推/转推/评论、关注/取关、获取推文与评论、搜索推文、@提及。
- **ticket api**: 工单/客服票务：创建/关闭/编辑/解决工单、查询工单、登录登出。
- **trading bot**: 股票交易与账户：下单/撤单、资金存取、自选股、行情与订单查询、按价格筛选、价格异动通知。
- **travel booking**: 旅行预订：航班预订与取消、信用卡与预算、机场与票价、预订历史、保险、发票、客服。
- **vehicle control**: 车辆控制：发动机、车门锁、空调、车灯、手刹/刹车踏板、定速巡航、油量、胎压、导航、车速与单位换算。
- **web search**: 网页搜索与抓取：按关键词与地区搜索、按 URL 拉取内容（raw/markdown/截断）。

# Target Domains (Retained if relevant to any of the listed domains)

- **gorilla file system**: Unix-like file system, covering directory navigation, creation, read/write and deletion of files and directories, as well as basic operations including `cat`, `cd`, `cp`, `diff`, `du`, `echo`, `find`, `grep`, `ls`, `mkdir`, `mv`, `pwd`, `rm`, `rmdir`, `sort`, `tail`, `touch`, `wc`, etc.
- **math api**: Mathematical operations and statistics: addition, subtraction, multiplication, division, absolute value, logarithm, power, square root, rounding, maximum/minimum value, mean, standard deviation, summation, percentage, and unit conversion (Imperial/SI).
- **memory kv**: Key-value memory: add, delete, modify and query operations for long-term and short-term memory, key-based retrieval, BM25+ similar key search, and memory clearing.
- **memory rec sum**: Recursive summary memory: append, replace, retrieve, full update or clearing of memory content.
- **memory vector**: Vector memory: add, delete, modify and query operations for entries, similarity-based retrieval (top_k), as well as long-term and short-term memory management.
- **message api**: Workspace messaging and contacts: contact adding, message sending/receiving/deletion/search, user list management, login, and access to sending history and statistics.
- **posting api**: Twitter posting and social interaction: authentication, tweeting/retweeting/commenting, following/unfollowing, fetching tweets and comments, tweet search, and @mentions.
- **ticket api**: Work order & customer service ticketing: work order creation/closure/editing/resolution, work order query, and login/logout.
- **trading bot**: Stock trading and account management: order placement/cancellation, fund deposit/withdrawal, watchlist management, market quote and order query, price-based filtering, and price fluctuation alerts.
- **travel booking**: Travel booking: flight booking and cancellation, credit card and budget management, airport and fare inquiry, booking history, insurance, invoice issuance, and customer service.
- **vehicle control**: Vehicle control: engine, door lock, air conditioning, vehicle lights, handbrake/brake pedal, cruise control, fuel level, tire pressure, navigation, vehicle speed and unit conversion.
- **web search**: Web search and crawling: keyword and region-based search, URL-based content fetching (raw/markdown/truncated).

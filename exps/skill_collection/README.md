# Skill Collection Experiment

Early-stage experiment for collecting and standardizing Skills (Anthropic-style tool definitions) into the Astra registry.

## Purpose

- Discover and ingest tool definitions from various sources
- Normalize them into the `astra.schemas.Skill` format
- Validate and register skills in `skillshub/`, then curate into `skills/`

# File Categories Reference

This reference provides a mapping of common file extensions to their respective categories/folders.

## Default Mappings

- **Images**: .jpg, .jpeg, .png, .gif, .bmp, .svg, .webp, .tiff
- **Documents**: .pdf, .doc, .docx, .txt, .rtf, .odt, .xls, .xlsx, .ppt, .pptx, .csv
- **Audio**: .mp3, .wav, .flac, .aac, .ogg, .m4a
- **Video**: .mp4, .mkv, .mov, .avi, .wmv, .flv, .webm
- **Archives**: .zip, .tar, .gz, .7z, .rar, .bz2
- **Code**: .py, .js, .ts, .html, .css, .cpp, .c, .h, .java, .go, .rs, .sh, .json, .yml, .yaml, .md
- **Executables**: .exe, .bin, .msi, .appimage, .deb, .rpm
- **Fonts**: .ttf, .otf, .woff, .woff2

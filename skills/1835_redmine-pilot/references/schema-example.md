# Redmine 数据结构示例

此文件是模板，请根据你的 Redmine 实例填写。

## 服务器
- URL: https://your-redmine-server.com
- 认证: API Key（从「我的账户」页面获取）

## 状态 (常用)

运行以下命令获取你的状态列表：
```bash
python3 scripts/redmine.py statuses
```

## 跟踪器 (常用)

运行以下命令获取你的跟踪器列表：
```bash
python3 scripts/redmine.py trackers
```

## 优先级

运行以下命令获取你的优先级列表：
```bash
python3 scripts/redmine.py priorities
```

## 项目

运行以下命令获取你的项目列表：
```bash
python3 scripts/redmine.py projects
```

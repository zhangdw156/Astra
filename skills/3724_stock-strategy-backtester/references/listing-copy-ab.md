# A/B Listing Copy

Use these variants for marketplace title/subtitle/description testing.

## Variant A (Professional Quant)

- Title:
  - Stock Backtest Analyst
- Short Description:
  - Backtest stock strategies for win rate and returns
- One-Line Pitch:
  - Validate stock strategies on historical OHLCV with win rate, CAGR, drawdown, Sharpe, and trade logs.
- Description:
  - Stock Backtest Analyst helps traders and researchers evaluate stock strategy quality with consistent, reproducible backtests.
  - It supports three core strategy families: SMA crossover (trend), RSI mean reversion, and breakout.
  - Every run includes transaction costs and slippage, and avoids look-ahead by executing at next-bar open.
  - Outputs include JSON for automation and CSV trade logs for audit.

## Variant B (Outcome-Focused)

- Title:
  - Win Rate & Return Backtester
- Short Description:
  - One-command stock backtesting with clear risk metrics
- One-Line Pitch:
  - See if your stock strategy really works before risking capital.
- Description:
  - Run a stock strategy backtest in one command and instantly get the numbers that matter:
  - Win rate, total return, CAGR, max drawdown, Sharpe ratio, profit factor, and trade count.
  - Compare parameter sets fast, keep assumptions explicit, and export trade logs for deep review.

- One-Line Pitch:
  - 用历史 OHLCV 数据快速评估策略胜率、收益率与回撤风险。
- Description:
  - 该技能面向量化研究与交易策略验证场景，支持均线、RSI、突破三类主流规则。

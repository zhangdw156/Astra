# Paper Check - 毕业论文格式规范检测与修复工具

[![版本](https://img.shields.io/badge/版本-1.0.0-blue.svg)](https://github.com)
[![Python](https://img.shields.io/badge/Python-3.7+-green.svg)](https://www.python.org/)
[![许可证](https://img.shields.io/badge/许可证-MIT-orange.svg)](LICENSE)

> 本科/研究生毕业论文格式规范检测、自动修复与对比报告生成工具

## 功能特点

- 🔍 **全面检测**：自动检测论文格式问题，包括页面设置、字体字号、段落格式、页眉页脚等
- 🔧 **自动修复**：一键修复常见格式问题，如页边距、缩进、行距、字体等
- 📊 **对比报告**：生成精美的修改前后对比报告，清晰展示所有修改内容
- 📚 **规范支持**：支持任意学校的论文撰写规范文档输入
- 🎯 **智能定位**：自动识别章节结构，精确定位修改位置

## 快速开始

### 安装依赖

```bash
pip install python-docx lxml pywin32  # pywin32 仅 Windows 需要
```

### 使用方法

在 Claude Code / WorkBuddy 中使用：

```
请帮我检查这篇论文的格式规范：论文.docx
```

或指定学校规范：

```
请按照这个规范文件检查论文格式：
规范文件：学校论文撰写规范.docx
论文文件：我的论文.docx
```

## 核心工作流

1. **规范解析**：解析学校提供的论文撰写规范文档
2. **格式检测**：全面扫描论文文档的格式属性
3. **问题诊断**：生成问题清单，标注严重程度
4. **自动修复**：修复可自动化的格式问题
5. **报告生成**：生成修改前后对比报告

## 支持的格式修复

| 类别 | 支持项 |
|------|--------|
| 页面设置 | ✅ 页边距、页眉距、页脚距、装订线 |
| 段落格式 | ✅ 首行缩进、悬挂缩进、行距、段间距 |
| 字体设置 | ✅ 中文字体、西文字体、字号、加粗 |
| 对齐方式 | ✅ 左对齐、居中、右对齐、两端对齐 |
| 特殊标题 | ✅ 摘要、目录、致谢、参考文献标题格式 |
| 表格格式 | ✅ 三线表边框 |
| 页眉页脚 | ✅ 页眉内容、字体、对齐 |

## 已知限制

由于 python-docx 库的限制，以下问题无法自动修复：

- ❌ 页眉横线样式
- ❌ 参考文献编号内容自动添加
- ❌ 目录自动更新
- ❌ 图片/公式的精确位置调整
- ❌ 脚注格式

## 内置默认规范

基于 GB/T 7713.1-2006《学位论文编写规则》和多校通用惯例：

- **页边距**：上 2.8cm / 下 2.2cm / 左 3.0cm / 右 2.0cm
- **页眉距**：1.8cm
- **页脚距**：1.4cm
- **正文字体**：小四号宋体 / Times New Roman
- **行距**：1.5 倍
- **首行缩进**：2 字符

## 技术要点

### 缩进处理

Word 文档中的首行缩进有两种单位：
- 固定厘米值（`firstLine`）：不随字号变化
- 字符单位（`firstLineChars`）：随字号自动调整（推荐）

### 函数执行顺序

修复脚本应按以下顺序执行：

```
页面设置 → 段落格式 → 参考文献 → 特殊标题 → 页眉
```

详细说明请参考 [SKILL.md](./SKILL.md) 中的"脚本设计注意事项"章节。

# LingJi运营工单分析Skill

> Lingji工单数据分析与运营监控 | 支持PDF报告生成

**作者**: ChengYusheng  
**版本**: 1.0.0  
**标签**: 运维, 工单分析, 数据报表, PDF生成

---

## 技能简介

本技能对Lingji平台的运维工单数据进行全方位分析，自动生成专业的运营分析报告。支持终端直接输出和PDF报告两种形式，方便发送和存档。

### 核心功能

- 📊 工单总量与分布统计
- ⏱️ 处理时效分析（平均时长、超时监控）
- 👥 支撑人员工作量与质量评估
- ⭐ 满意度分析（含有效评价过滤）
- 🔥 热点问题智能识别
- 📑 一键生成PDF分析报告

## 数据结构

### 输入格式

原始Excel工单包含24个字段：

| 字段类别 | 包含字段 |
|---------|---------|
| 基本信息 | 提单人、提单人所属公司、工单单号、工单类型 |
| 时间信息 | 创建时间、认领时间、响应时间、归档时间、处理时长 |
| 状态信息 | 工单状态、菜单、模块、归属域 |
| 内容信息 | 工单内容描述、处理人归属公司、开发域版本 |
| 流转信息 | 处理工作组、当前处理人、历史处理人 |
| 处理信息 | 红军处理意见描述 |
| 评价信息 | 满意度选项、满意度分值 |
| 超时信息 | 响应超期信息、响应超时状态 |

### 关键字段枚举

**工单状态：**
- `红军已完成` - 处理人已回复，等待用户确认
- `待用户确认` - 用户尚未确认解决方案
- `蓝军已完成` - 二线已复核
- `待蓝军处理` - 需要二线介入

**工单类型：**
- 问题咨询（最多）
- 普通投诉
- 意见反馈
- 升级投诉

**满意度分值：**
| 评价 | 分值 |
|-----|-----|
| 非常满意 | 10分 |
| 满意 | 8分 |
| 一般 | 6分 |
| 不满意 | 4分 |
| 非常不满意 | 2分 |
| 未评价 | 0分 |

## 分析维度

### 1. 整体运营概览

- 工单总量与类型分布
- 工单状态分布（已完成/待确认）
- 归属域分布（管理域/开发域）
- 满意度分布与平均分

### 2. 时效性分析

- 平均处理时长（含中位数）
- 处理时长分位数统计
- 响应超时工单数量与占比
- 超期工单明细清单

### 3. 支撑人员分析

按处理工作组和具体处理人统计：
- 处理工单数量
- 已完成率
- 满意度评分（仅计算有效评分）
- 平均处理时长
- 响应超时情况

### 4. 热点问题分析

通过对工单内容描述的文本分析，自动识别问题类型：

| 问题类型 | 占比 | 典型场景 |
|---------|-----|---------|
| 流程审批 | 4.5% | 审批流、流程节点、流程配置 |
| 需求管理 | 2.4% | 需求创建、需求变更、需求查询 |
| 系统报错 | 1.4% | 各类系统错误提示 |
| 立项/需求关联 | 9% | 项目立项、需求挂载、关联配置 |
| 权限问题 | 8.8% | 权限不足、无法查看、访问受限 |
| 数据问题 | 7.7% | 数据错误、数据缺失、数据不同步 |
| 删除问题 | 6.7% | 无法删除、删除报错 |
| 制品/仓库 | 6.4% | 制品上传、仓库管理 |
| 建议/优化需求 | 6.4% | 功能建议、系统优化 |
| 研发计划 | 4.8% | 计划填报、计划下达 |
| 成果管理 | 4.3% | 成果申报、成果转化 |
| 预算问题 | 2.4% | 预算配置、预算调整 |
| 登录问题 | 2.2% | 登录异常、登录失败 |
| 证书问题 | 1.2% | CA证书、安装配置 |

### 5. 质量专项分析

- 回复内容长度分析
- 标准化回复检查
- 复杂问题处理时效

## 使用方法

### 环境准备

```bash
# 激活虚拟环境
source ~/.openclaw/workspace/.venv/bin/activate

# 安装依赖
pip install pandas openpyxl playwright
playwright install chromium
```

### 生成PDF报告（推荐）

```bash
# 使用Playwright生成PDF（中文显示最佳）
python3 scripts/generate_pdf.py <Excel文件> <输出PDF文件>

# 示例
python3 scripts/generate_pdf.py 全量工单导出.xlsx 工单分析报告.pdf
```

### 终端直接分析

```bash
# 完整分析报告（终端输出）
python3 scripts/lingji_gongdan.py report <excel文件>
```

## 注意事项

1. **数据格式**：输入文件应为Excel格式(.xlsx)
2. **字段校验**：确保字段名称与模板一致
3. **满意度计算**：自动过滤未评价（0分）的数据
4. **时间单位**：处理时长单位为分钟
5. **PDF依赖**：首次使用需运行 `playwright install chromium`

---

## 维护建议

- **定期分析**：建议每月导出数据后进行分析
- **重点关注**：满意度持续低于均值的处理人
- **超时跟进**：识别超时工单并及时跟进处理
- **产品反馈**：热点问题汇总后反馈产品改进
- **报告存档**：PDF报告可直接通过微信/WhatsApp发送

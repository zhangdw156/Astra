# Content Repurposing Templates

## From Blog Post

**Original:** 1000-word article

**Create:**
```
Twitter Thread:
- Tweet 1: Hook + main thesis
- Tweet 2-5: Key points (one per tweet)
- Tweet 6: Summary + CTA

LinkedIn Post:
- Hook (first 2 lines visible)
- 3-5 key takeaways
- Link to full article
- 3-5 hashtags

Instagram:
- Carousel: 5-7 slides with key points
- Caption: Summary + question for engagement
- 10-15 hashtags
```

## From Video/Podcast

**Original:** 30-minute video

**Create:**
```
Quote Clips:
- 3-5 standout quotes as images
- Post across all platforms

Key Takeaways:
- Bullet list of main points
- Twitter thread or LinkedIn post

Behind-the-Scenes:
- Setup photo + story
- Instagram Stories or Twitter

Transcript Snippets:
- Turn best moments into text posts
- LinkedIn articles or Twitter threads
```

## From Product Launch

**Original:** New product/service

**Create:**
```
Teaser (1 week before):
- "Something big is coming..."
- Mysterious hint
- Countdown

Announcement (Day 0):
- What it is
- Why it matters
- How to get it
- Launch offer

Feature Highlights (Days 1-7):
- One feature per day
- Problem → Solution format
- User benefit focused

Social Proof (Ongoing):
- Early user testimonials
- Use case examples
- Results/metrics
```

## From One Post → Many Platforms

**Original:** Any piece of content

**Adaptation Formula:**
```
1. Extract core message (1 sentence)
2. Identify 3-5 key points
3. For each platform:
   - Twitter: Punchy, 280 chars, 2-3 hashtags
   - LinkedIn: Professional, story-driven, 3-5 hashtags
   - Instagram: Visual + caption, 10-15 hashtags
   - Facebook: Conversational, 1-3 hashtags
```
